# 📸 Computer Vision Photo Editor

> ⚠️ This is a **Computer Vision (CV)** project using OpenCV .

An interactive web-based photo editor built using **Streamlit, OpenCV, and Python**.  
This application allows users to upload or capture images and apply real-time image processing techniques.

---

## 🧠 Project Type
- Computer Vision (CV)
- Image Processing
- Real-time Image Editing


---

## 🚀 Features

- 📂 Upload image from system  
- 📷 Capture image using camera  
- 📏 Resize image  
- ☀️ Adjust brightness  
- 🎚️ Adjust contrast  
- 🎨 Apply filters:
  - Grayscale  
  - Blur  
  - Warm effect  
  - Sharpen  
  - Edge detection  
  - Sketch effect  
- 🔄 Side-by-side image comparison  
- 📥 Download edited image  

---

## 🛠️ Technologies Used

- Python  
- Streamlit  
- OpenCV  
- NumPy  
- PIL (Pillow)  

---

## 🌐 Browser Support

- ✅ Works best in Google Chrome  
- ⚠️ If camera does not work in Edge, try using Chrome
